To compile the text UI version:
javac SystemMain.java

To run the text UI version:
java SystemMain

To compile the GUI version:
javac --module-path "PATH TO JAVAFX LIB" --add-modules javafx.controls,javafx.fxml HotelGUI.java
In my case, javac --module-path "C:\javafx\javafx-sdk-17.0.1\lib" --add-modules javafx.controls,javafx.fxml HotelGUI.java

To run the GUI version:
java --module-path "PATH TO JAVAFX LIB" --add-modules javafx.controls,javafx.fxml HotelGUI
In my case, java --module-path "C:\javafx\javafx-sdk-17.0.1\lib" --add-modules javafx.controls,javafx.fxml HotelGUI
